## Detailed Command Mapping

| Directory | Corresponding Output | Description | Corresponding Nix Command |
| :--- | :--- | :--- | :--- |
| [`hosts/`](/en/manual-hosts) | `nixosConfigurations.<name>` | System Configuration Entry | `nixos-rebuild switch --flake .#<name>` |
| [`modules/`](/en/manual-modules) | `nixosModules.<name>` | NixOS Modules | - (Usually referenced by hosts) |
| [`pkgs/`](/en/manual-pkgs) | `packages.<system>.<name>` | Packages | `nix build .#<name>` |
| [`apps/`](/en/manual-apps) | `apps.<system>.<name>` | Applications | `nix run .#<name>` |
| [`checks/`](/en/manual-checks) | `checks.<system>.<name>` | Checks | `nix flake check .#<name>` |
| [`shells/`](/en/manual-shells) | `devShells.<system>.<name>` | Dev Shells | `nix develop .#<name>` |
| [`lib/`](/en/manual-lib) | `lib.<name>` | Library | `nix eval .#lib.<name>` |
| [`templates/`](/en/manual-templates) | `templates.<name>` | Project Templates | `nix flake init --template .#<name>` |
