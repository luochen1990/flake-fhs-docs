## 1 Minute Tutorial

<Steps>

1.  **Initialize Project**

    Flake FHS provides two initialization templates:

    <Tabs>
      <TabItem label="Pure Nix Project" icon="rocket">
        Suitable as a **nixos-config** or **pure Nix package repository**.

        ```bash
        nix flake init --template github:luochen1990/flake-fhs
        ```

        It generates a `flake.nix` file, and you can continue to add standard directory structure like `pkgs/`, `hosts/`, `modules/`.
      </TabItem>

      <TabItem label="Embed in Non-Nix Project" icon="puzzle">
        Suitable for adding Nix support to **non-pure Nix projects** (such as Python, Node.js, Rust).

        ```bash
        nix flake init --template github:luochen1990/flake-fhs#embed
        ```

        It generates a `flake.nix` file and `nix/` directory, minimizing interference with the existing project structure. You can continue to add standard directory structure like `pkgs/`, `hosts/`, `modules/` under the `nix/` directory.
      </TabItem>
    </Tabs>

2.  **Add a Package**

    Create file `pkgs/hello.nix`:

    ```nix
    { writeShellScriptBin }:
    writeShellScriptBin "hello" ''
      echo "Hello, Flake FHS!"
    ''
    ```

    *No need to manually modify `flake.nix`, the framework will automatically discover it.*

3.  **Build and Run**

    ```bash
    nix build .#hello
    nix run .#hello
    ```

</Steps>
