## 1分钟入门 (Tutorial)

<Steps>

1.  **初始化项目**

    Flake FHS 提供两种初始化模板：

    <Tabs>
      <TabItem label="纯 Nix 项目" icon="rocket">
        适用于作为 **nixos-config** 或**纯 Nix 软件包仓库**。

        ```bash
        nix flake init --template github:luochen1990/flake-fhs
        ```

        它会生成 `flake.nix` 文件，你可以继续添加 `pkgs/`, `hosts/`, `modules/` 等标准目录结构。
      </TabItem>

      <TabItem label="嵌入到非 Nix 项目" icon="puzzle">
        适用于为**非纯 Nix 项目** (如 Python, Node.js, Rust) 添加 Nix 支持。

        ```bash
        nix flake init --template github:luochen1990/flake-fhs#embed
        ```

        它会生成 `flake.nix` 文件和 `nix/` 目录，尽可能不干扰现有的项目结构。你可以继续在 `nix/` 目录下添加 `pkgs/`, `hosts/`, `modules/` 等标准目录结构。
      </TabItem>
    </Tabs>

2.  **添加一个包**

    创建文件 `pkgs/hello.nix`：

    ```nix
    { writeShellScriptBin }:
    writeShellScriptBin "hello" ''
      echo "Hello, Flake FHS!"
    ''
    ```

    *无需手动修改 `flake.nix`，框架会自动发现它。*

3.  **构建与运行**

    ```bash
    nix build .#hello
    nix run .#hello
    ```

</Steps>
