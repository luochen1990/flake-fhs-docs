## 1分钟入门 (Tutorial)

<Tabs>
  <TabItem label="纯 Nix 项目" icon="rocket">

适用于作为 **nixos-config** 或**纯 Nix 软件包仓库**。

<Steps>

1. **初始化项目**

   ```bash
   nix flake init --template github:luochen1990/flake-fhs
   ```

   它会生成 `flake.nix` 文件，你可以继续添加 `pkgs/`, `hosts/`, `modules/` 等标准目录结构。

2. **添加一个 Host**

   创建文件 `hosts/laptop/configuration.nix`：

   ```nix
   { config, lib, pkgs, ... }:
   {
     # 注意：这只是配置的一部分，完整配置还需包含 boot loader、filesystem 等内容
     system.stateVersion = "26.05";
   }
   ```

   *无需手动修改 `flake.nix`，框架会自动发现它。记得 `git add` 新文件，否则 flake 无法识别。*

3. **添加一个 Module**

   创建文件 `modules/tweaks.nix`：

   ```nix
   { config, lib, ... }:
   {
     options.tweaks.enable = lib.mkEnableOption "system tweaks";

     config = lib.mkIf config.tweaks.enable {
       # 启用 flakes 和 nix-command
       nix.settings.experimental-features = [ "nix-command" "flakes" ];
       # 将 Caps 映射为 Ctrl
       services.xserver.xkb.options = "ctrl:nocaps";
     };
   }
   ```

4. **在 Host 中启用 Module**

   修改 `hosts/laptop/configuration.nix`：

   ```nix
   { config, lib, pkgs, ... }:
   {
     # 注意：这只是配置的一部分，完整配置还需包含 boot loader、filesystem 等内容
     system.stateVersion = "26.05";

     tweaks.enable = true;  # 启用自定义 module
   }
   ```

   *框架会自动 import 所有 modules，无需手动添加 imports。*

5. **构建验证**

   ```bash
   nixos-rebuild build --flake .#laptop
   ```

</Steps>

  </TabItem>

  <TabItem label="嵌入到非 Nix 项目" icon="puzzle">

适用于为**非纯 Nix 项目** (如 Python, Node.js, Rust) 添加 Nix 支持。

<Steps>

1. **初始化项目**

   ```bash
   nix flake init --template github:luochen1990/flake-fhs#embed
   ```

   它会生成 `flake.nix` 文件和 `nix/` 目录，尽可能不干扰现有的项目结构。

2. **添加一个包**

   创建文件 `nix/pkgs/weather.nix`：

   ```nix
   { writeShellScriptBin, curl }:
   writeShellScriptBin "weather" ''
    ${curl}/bin/curl -s wttr.in/?format=3
   ''
   ```

   *无需手动修改 `flake.nix`，框架会自动发现它。记得 `git add` 新文件，否则 flake 无法识别。*

3. **构建与运行**

   ```bash
   nix build .#weather
   nix run .#weather
   ```

</Steps>
